Building this Steam Companion App was honestly a huge learning experience for me, especially when it came to tying React, routing, and serverless functions together. The biggest headache by far was figuring out Steam's login system. Instead of the modern OAuth setups most sites use today, Steam still uses an older OpenID protocol with insanely strict firewalls that kept blocking my local testing. I spent a lot of time just trying to get the app to talk to Steam without getting flagged as a bot, eventually having to use a mix of client-side redirects and Netlify functions to finally get the handshake working. It was definitely frustrating at times, but finally seeing that login work was a great feeling.

Another big lesson was learning to code defensively. Since Steam defaults most user profiles to private, I couldn't just assume the data would always be there. I had to build guardrails and fallback UI screens so the app wouldn't just crash when it tried to read a hidden profile. In the future, I'd love to hook this app up to a real database like Firebase so users can actually "pin" the random achievements they get and save them to a persistent to-do list. I’d also love to add a "Friends" tab to compare playtimes. Overall, this project pushed me out of tutorial mode and forced me to build something real with a notoriously stubborn API.
